Data visualisation style guide

This is a style guide prototype for data visualisation work within DWP. It gives a generic overview but also provides information on how to apply it in within tools such as Microsoft Power BI (PBI), and Qlik.

All styles are detailed to be as consistent as possible with the GOV.UK design system, or as close as possible where there is an existing style or pattern.
