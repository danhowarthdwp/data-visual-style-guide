Graphs

Graphs titles, intro text and container

Creating a custom text box for the graph title / heading will allow custom spacing and create room to add introcuduction text to help give context to the graph.

Use a Medium 16pt heading for the title with standard 12pt font for the intro text.

Ensure both text boxes line up consistenly within the container using the General, X Position.

Background shape / graph container:

Fill: Off
Outline: #B1B4B6
Outline weight: 1pt
