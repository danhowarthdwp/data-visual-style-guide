Navigation

Taken from the DWP design examples for secondary navigation.

There are multiple parts to this pattern, using buttons and layered shapes. It helps to rename these shapes something valuable when layering in the View > Selection pane.

Active nav button

Select Button > Blank
Outline: Off Text: vertical alignment: Middle
Horizontal alignment: cente
Margins: 4px
Icon: Off
Background: off
Action: off (this isn't needed as this has to look like the active selection)

Active nav button underline (layer ontop of the active nav button and grey underline)

Shape> line
Outline: On
Outline colour: #005EA5
Outline weight: 2pt

Other nav link buttons

Select Button > Blank
Outline: Off Text: font colour: #005EA5 Vertical alignment: Middle
Horizontal alignment: cente
Margins: 4px
Icon: Off
Background: off
Action: on
Type: Page navigation
Destination: Select your target page

Grey underline border (bottom layer)

Shape > line Online: On
Title: off
Online colour: #DEE0E2
Outline weight: 1pt
