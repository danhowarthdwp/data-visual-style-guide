Text box alignment

When creating content on the left side of the screen make sure it's all aligned evenly by setting each component in the General section, X position: 30.

Visual interactions

PBI is an interactive tool that updates values when visuals are interacted with. For example a data point can be selected on a graph and update the values in a table to show data on a perticular date.

To turn this function off select 'Visual interactions' from the header ribbon > Edit interactions: On and select which visuals you want to turn off interactivity.
